# 问答数据分析工作记录

## 数据来源

文件：`问答.xlsx`
平台：电商平台买家社区问答（商品维度）

---

## 数据结构

| 列名 | 说明 |
|------|------|
| `auctionNumId` | 商品编号 |
| `date` | 提问时间（含时分秒） |
| `questionId` | 问题唯一ID |
| `question` | 买家提问内容 |
| `answer_date` | 回答日期 |
| `answer_nickname` | 回答者昵称（已脱敏） |
| `answer` | 回答内容 |
| `时间是否在范围` | 布尔值，标记是否在目标时间范围内 |

**数据规模：** 62,560 行（含表头），62,559 条记录，共 99 个不同商品

`时间是否在范围` 分布：
- `True`（在范围内）：21,783 条（约 34.8%）
- `False`（不在范围内）：40,776 条（约 65.2%）

---

## 分析任务

### 目标

筛选满足以下条件的商品：

- 提问时间（`date`）在 **2022-10-01 ~ 2024-09-30** 两年窗口内
- 以 **2023年10月** 为处理节点，处理前后各 4 个季度（共 8 个季度）
- 每个季度均至少存在 **1 条**问答记录

### 季度划分

| 阶段 | 标签 | 时间范围 |
|------|------|----------|
| 处理前 | pre_Q1 | 2022-10-01 ~ 2022-12-31 |
| 处理前 | pre_Q2 | 2023-01-01 ~ 2023-03-31 |
| 处理前 | pre_Q3 | 2023-04-01 ~ 2023-06-30 |
| 处理前 | pre_Q4 | 2023-07-01 ~ 2023-09-30 |
| 处理后 | post_Q1 | 2023-10-01 ~ 2023-12-31 |
| 处理后 | post_Q2 | 2024-01-01 ~ 2024-03-31 |
| 处理后 | post_Q3 | 2024-04-01 ~ 2024-06-30 |
| 处理后 | post_Q4 | 2024-07-01 ~ 2024-09-30 |

### 结果

- 数据集商品总数：**99**
- **满足条件的商品数：66**
- 被排除商品数：33（至少有一个季度无记录）

## 分析脚本

脚本文件：`analyze_qa.py`

依赖：`openpyxl`（`pip install openpyxl`）

运行方式：
```bash
python3 analyze_qa.py
```


- 红色虚线：处理节点（2023年10月）
- 蓝色底色区域：处理前；橙色底色区域：处理后
